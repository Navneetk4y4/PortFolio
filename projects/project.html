Read more Auto-Forensic is an automated cybersecurity tool designed to perform comprehensive system investigations by analyzing both running processes and network connections. It collects detailed process information including executable paths, command lines, and file hashes, which are then scanned against VirusTotal's threat database. The tool simultaneously monitors network activity, identifying suspicious connections and checking IP reputations using AbuseIPDB. By generating structured CSV reports, it provides security professionals with actionable forensic data to detect potential compromises. This Python-based solution is particularly valuable for post-incident analysis and routine security audits on Windows systems.

Project Description

This tool helps investigate systems after potential attacks by:

  • Collecting all running processes and their executable paths
  • Dumping network connections into structured files
  • Generating file hashes and analyzing them via VirusTotal API
  • Checking connection IPs against AbuseIPDB for reputation
  • Creating comprehensive forensic reports

Key Features

Network Analysis

Identifies all active connections and flags suspicious IPs

Process Scanning

Detects potentially malicious processes and executables

Threat Intelligence

Integrates with VirusTotal and AbuseIPDB APIs

Comprehensive Reports

Generates detailed CSV reports for analysis

import csv
import subprocess
import re
import os
import requests
from dotenv import load_dotenv
import pandas as pd

# Load API key from .env file
load_dotenv()
API_KEY = os.getenv("ABUSEIP_API_KEY")

def get_network_connections():
    """Get all network connections using netstat -ano"""
    try:
        result = subprocess.check_output(
            ['netstat', '-ano'],
            text=True,
            stderr=subprocess.STDOUT,
            shell=True
        )
        return result
    except subprocess.CalledProcessError as e:
        print(f"Command failed: {e}")
        return None

Installation & Usage

1. Clone Repository

Terminal 
# Clone the repository
git clone https://github.com/Navneetk4y4/auto-forensic.git
cd auto-forensic

2. Setup Environment

Terminal 
# Check Python version (3.8+ required)
python --version

# Install dependencies
pip install -r requirements.txt

3. Configuration

Create a .env file in the project root with your API keys:

.env 
# VirusTotal API Key (required for process analysis)
VIRUSTOTAL_API_KEY="your_virustotal_key_here"

# AbuseIPDB API Key (required for network analysis)
ABUSEIP_API_KEY="your_abuseipdb_key_here"

4. Running the Tool

Terminal 
# Full system scan (default)
python main.py

# Network analysis only
python main.py --network-only

# Process analysis only
python main.py --process-only

# View all options
python main.py --help

5. Generated Reports

Network Reports

  • network_connections.csv - All active connections
  • foreign_addresses.csv - Unique foreign IPs
  • abuse_report.csv - IP reputation data

Process Reports

  • process_report.csv - Process details with VirusTotal results
